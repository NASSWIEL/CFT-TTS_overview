
This repository provides a set of audio samples from the **Blizzard 2023 segmented corpus**, including:

- **Reference recordings**
- **ZipVoice outputs**
- **ZipVoice 24kHz outputs**
- **Fine-tuned ZipVoice outputs**



## Directory Structure
Structure:
```
collected_samples/
├── AD/
│   ├── reference/
│   ├── zipVoice/
│   ├── zipVoicekhz24/
│   └── zipVoic_fintuned/
└── NEB/
    ├── reference/
    ├── zipVoice/
    ├── zipVoicekhz24/
    └── zipVoic_fintuned/
```
